# 🧠 AI Memory System

> 基于文档的AI协作系统，让AI具备长期记忆能力

[![License](https://img.shields.io/badge/license-Apache--2.0-blue.svg)](LICENSE)
[![Platform](https://img.shields.io/badge/platform-macOS%20%7C%20Linux%20%7C%20Windows-lightgrey)](https://github.com/shamcleren/ai_memory)
[![AI Tool](https://img.shields.io/badge/AI%20Tool-Cursor%20%7C%20VS%20Code%20%7C%20Claude-orange)](https://cursor.sh/)

**AI Memory System** 是一个基于文档的AI协作系统，旨在帮助开发者与AI助手建立长期、连续、智能的协作关系。通过文档化、规则化、自动化的方式，让AI能够记住你的工作习惯、管理你的知识资产、保持上下文连续性，并主动提供帮助。

## ✨ 核心特性

- 🧠 **长期记忆** - 突破单次对话限制，跨会话记忆连续性
- 📝 **知识管理** - 自动创建、关联、索引技术文档和项目知识
- ✅ **智能任务** - 自然语言创建任务，自动判断优先级
- 🔗 **知识关联** - 自动建立文档关联，形成知识网络
- 🚀 **开箱即用** - 一句话告诉AI，自动构建完整系统
- 📊 **持续进化** - AI从使用中学习，不断优化协作效果

## 🚀 快速开始

### ⚠️ 前置条件：需要 AI 工具

**重要**：在使用本系统前，必须先安装支持 AI 功能的工具。

#### 推荐工具：Cursor

**什么是 Cursor？**
- Cursor 是一个基于 VS Code 的 AI 代码编辑器
- 内置强大的 AI 助手，可以理解你的需求并自动执行操作
- 完全免费使用

**如何安装？**
1. 访问 https://cursor.sh/
2. 下载对应系统的安装包（Windows/Mac/Linux）
3. 安装并打开 Cursor
4. 在 Cursor 中打开你的工作空间目录

**其他选择**：
- VS Code + GitHub Copilot
- Claude Desktop
- 其他支持文件操作的 AI 工具

**为什么需要 AI 工具？**
本系统的核心是**与 AI 协作**，AI 会自动帮你：
- 创建目录结构和文档
- 根据模板生成规则文档
- 理解需求并执行操作
- 管理任务和生成日报

**没有 AI 工具，系统无法自动构建！**

---

### 第一步：告诉AI构建系统

**前提**：确保你已经在 Cursor 中打开了工作空间目录。

**如何与 AI 对话？**
- 在 Cursor 中，按 `Cmd/Ctrl + L` 打开 AI 聊天窗口
- 或者点击界面上的 AI 图标
- 在聊天窗口中输入下面的指令

**只需一句话告诉AI**：

```
我想构建一个AI记忆系统，目标工作空间根目录是：/你的/工作空间/路径

请将生成的AI记忆系统相关目录（如`🤖 AI记忆系统/`）添加到目标工作空间根目录的.gitignore文件中

请读取当前目录下的文档，根据"AI记忆系统开箱即用模板.md"在目标工作空间根目录自动创建所有需要的目录、规则文档和脚本。
```

**说明**：
- 将 `/你的/工作空间/路径` 替换为你希望生成系统的目标目录路径
- AI 会自动读取模板文档，理解系统结构
- 然后在**指定的目标工作空间根目录**自动创建所有需要的文件和目录
- **重要**：AI会在你指定的路径下生成整个AI记忆系统，而不是在当前打开的目录

**可选：配置代码库地址**（推荐开发场景）

如果你有代码库需要关联，可以在构建时一并告诉AI：

```
我想构建一个AI记忆系统，目标工作空间根目录是：/你的/工作空间/路径
我的代码库地址是：https://github.com/username/repo 或 /本地/代码库/路径

请将生成的AI记忆系统相关目录（如`🤖 AI记忆系统/`）添加到目标工作空间根目录的.gitignore文件中

请读取当前目录下的文档，根据"AI记忆系统开箱即用模板.md"在目标工作空间根目录自动创建所有需要的目录、规则文档和脚本。
```

AI会自动：
1. 读取 `AI记忆系统开箱即用模板.md` 了解完整结构
2. 读取 `AI协作方法论与设计理念.md` 理解设计理念
3. 根据模板自动创建所有目录结构
4. 根据模板自动创建所有规则文档
5. 根据模板自动创建所有脚本文件
6. 配置脚本路径
7. （如果提供）记录代码库地址到系统配置中

**完全不需要手动创建任何东西！**

### 第二步：开始使用

系统构建完成后，直接使用自然语言：
- "今天需要完成XX任务" → AI自动创建日报并添加任务
- "创建一个关于XX的技术方案文档" → AI创建文档并建议关联
- "XX任务已完成" → AI更新日报
